# Project Description

## Overview

This project is a comparative data analysis of income inequality, poverty, and welfare systems in Norway, the United States, and the Philippines. The goal is to examine how income distribution and material well-being differ across the three countries, and how public services and welfare systems shape the gap between rich and poor.

The project is designed to demonstrate practical skills in data collection, cleaning, analysis, visualization, and interpretation using real-world public data.

## Motivation

I wanted to build a portfolio project that shows more than just technical ability. This project combines data analysis with a broader social and economic context, showing that I can work with official statistics and turn them into clear, meaningful insights.

The topic is especially relevant because inequality and poverty are not only statistical problems, but also policy questions. By comparing different countries and welfare systems, the project can show how institutions and public services influence outcomes for households and individuals.

## Research Questions

The project is guided by the following questions:

- How does income distribution vary between Norway, the United States, and the Philippines?
- What do the data say about the gap between rich and poor?
- How do poverty levels differ across the three countries?
- What role do welfare systems and public services play in reducing inequality and poverty?

## Data Sources

The project uses official and research-based sources, including:

- Statistics Norway (SSB)
- U.S. Census Bureau
- Philippine Statistics Authority (PSA)
- Supporting research on the effects of public services and welfare on inequality and poverty

These sources provide comparable indicators such as Gini coefficients, poverty rates, and income distribution measures. In Norway, the research literature also shows that including the value of public services lowers measured inequality and poverty.

## Method

The analysis is carried out in several steps:

1. Collect and review the source datasets.
2. Clean and standardize the data.
3. Compare inequality and poverty indicators across countries.
4. Visualize the results using charts and summary tables.
5. Interpret the findings in the context of welfare policy and material living conditions.

Where possible, the project uses consistent measures across countries. When indicators are not directly comparable, the analysis focuses on trends and patterns rather than exact one-to-one comparisons.

## Expected Output

The final project will include:

- Cleaned datasets.
- Analytical tables.
- Visualizations and dashboard-style figures.
- A short written interpretation of the results.
- A portfolio-ready GitHub repository that shows practical data analysis skills.

## Project Value

This project is useful because it shows how public data can be turned into insight. It demonstrates skills that are relevant for data analysis, data science, and related analytical roles, including working with real datasets, explaining trends, and communicating findings clearly.

It also shows an ability to handle a topic that matters in real life: how inequality, poverty, and welfare systems affect people differently across countries.
