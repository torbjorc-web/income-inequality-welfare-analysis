# Data Dictionary

This document summarizes the main analysis tables used by the project and dashboard.

## norway_inequality_indicators_clean

- `year`: Observation year.
- `gini_all_population`: Gini coefficient for all households.
- `p90_p10_all_population`: P90/P10 ratio for all households.
- `s80_s20_all_population`: S80/S20 income ratio for all households.
- `gini_excl_student_households`: Gini coefficient excluding student households.
- `p90_p10_excl_student_households`: P90/P10 ratio excluding student households.
- `s80_s20_excl_student_households`: S80/S20 ratio excluding student households.

## usa_clean

- `income_type`: Income concept group from source (for example MONEY INCOME).
- `group_name`: Source grouping label.
- `measure`: Indicator name (for example Gini index of income inequality).
- `year_2023_estimate`: 2023 estimate value.
- `year_2023_moe`: 2023 margin of error.
- `year_2024_estimate`: 2024 estimate value.
- `year_2024_moe`: 2024 margin of error.
- `pct_change_estimate`: Change estimate between years in source table.
- `pct_change_moe`: Margin of error for percent change.

## philippines_clean

- `region`: Region label (includes national row `Philippines`).
- `gini_2009`: Regional Gini in 2009.
- `gini_2012`: Regional Gini in 2012.
- `gini_2015`: Regional Gini in 2015.
- `gini_2018`: Regional Gini in 2018.
- `gini_2021`: Regional Gini in 2021.
- `gini_2023`: Regional Gini in 2023.

## user_country_indicators

Rows uploaded from the dashboard CSV import flow.

- `country`: Country name.
- `year`: Observation year.
- `gini`: Gini coefficient in 0 to 1 scale.
- `p90_p10`: Optional P90/P10 ratio.
- `s80_s20`: Optional S80/S20 ratio.
- `welfare_proxy_value`: Optional welfare context metric value.
- `welfare_proxy_label`: Label for welfare metric.
- `source`: Optional source text.
- `notes`: Optional notes.
- `created_at`: Insert timestamp generated by SQLite.

## Notes

- Values are harmonized for analysis where feasible, but source definitions can differ across countries.
- Cross-country comparisons are strongest for broad trend direction and relative ranking.
